Portable Mode

By default, Blish HUD stores it's settings, modules, etc. under Documents\Guild Wars 2\addons\blishhud\. If you would prefer to keep Blish HUD's settings in the same folder as Blish HUD, you can use portable mode.

Instead of specifying the --settings argument as described in Launch Options a Settings folder can be created directly in the Blish HUD folder (next to Blish HUD.exe). If present, and no --settings argument is provided, Blish HUD will store all modules and data in this folder instead of the default under Documents.
