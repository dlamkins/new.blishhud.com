Description​

Allows you to show text on screen while the user is in range of the marker.

AttributeData TypeExampleDescription
infoarbitrary textinfo="Collect trinkets to complete the heart.&#xA;Deliver trinkets: 30%&#xA;Rescue pirates: 17%"The text shown on screen when the character is in range.
inforangefloatinforange="14.7"The distance away that the info text will begin showing on screen.
Multi-line info

XML's escaped newline &#xA; is supported and will allow you to show messages which intentionally span multiple lines in the info.

Affected by​

AttributeHow
TriggerRangeinforange acts as an alias for triggerrange in Blish HUD's Pathing module.
